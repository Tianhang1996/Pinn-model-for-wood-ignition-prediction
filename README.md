This database aims to facilitate the application of AI algorithms for wood ignition time prediction, and it is affiliated to the following paper.

S. Lin, H. Lin, M. J Gollner, T. Zhang, X. Huang (2025) Predicting Wood Ignition Time based on Multi-Source Database: Comparative of Data-Driven and Physics-Informed Models. (submitted to ProCI)

All the data are stored in the spreadsheet.
